# Caso de Estudio: Air Patagonia
Este proyecto consiste en el análisis integral de un sistema para el registro y despacho de bultos en una aerolínea local.

## El Problema
El sistema debe gestionar la recepción de pasajeros, la validación de boletos y el pesaje de equipaje, asegurando que los bultos no superen los límites permitidos y se despachen únicamente para vuelos dentro de una ventana de dos horas.

## Aspectos Destacados del Análisis
**Validación de Reglas de Negocio:** Implementación de alarmas lumínicas ante excesos de peso y restricciones horarias estrictas.
**Diseño de Interfaz:** Mockup detallado que incluye la selección de vuelos y carga dinámica de bultos.
**Ciclo de Vida del Objeto:** Diagrama de estados que modela el flujo desde que el bulto es "Iniciado" hasta que es "Aceptado" o "Rechazado".
